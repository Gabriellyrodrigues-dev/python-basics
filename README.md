# Fundamentos de Python

Repositório com exercícios e exemplos desenvolvidos em Python,
voltados à prática de lógica de programação e fundamentos da linguagem.

---

## Conteúdos abordados
- Variáveis e tipos de dados  
- Estruturas condicionais  
- Laços de repetição  
- Funções  

---

## Objetivo
Consolidar conhecimentos básicos em Python por meio de exercícios
práticos e estudo contínuo.
## Estrutura do repositório
- `variaveis.py` — Variáveis e tipos de dados
- `condicionais.py` — Estruturas condicionais (if/elif/else)
- `loops.py` — Laços de repetição (for/while)
- `funcoes.py` — Funções (def/return)
